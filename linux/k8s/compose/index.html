docker-compose

linuxk8sdocker
  • docker compose
  • docker-engine
  • install

安装

git下载地址:https://github.com/docker/compose/releases

推荐pip安装

使用

build

指定Dockerfile 文件,compose会利用它自动构建

command

覆盖容器启动后默认命令

links

链接其它服务的容器

ports

暴露端口信息给宿主机,使用(host:container) 必须字符串格式,yaml解析涉及进制

volumes

挂载路径,宿主机(host:container);上访模式(host:container:ro)

volumes_from

挂载容器或服务

devices

设配映射列表

depends_on

express之间依赖关系,

  • docker-compose up 按照依赖顺序启动

labels

向docker容器添加元数据

其它

docker run 支持

devops

核心原则是自动化、持续交付和跨职能合作

工具链全图

Build

在SCM的模块中：Git系列用的比较多，如Gitlab；

在CI模块中：Jenkins显然是最流行的；

在Build模块中：Maven、docker用的较多；

Test

在Testing模块中：Junit、Jmeter用的较多；Selenium

Deploy

在配置管理模块中：Puppet，Ansible用的比较多

Run

在Cloud/IaaS/PaaS模块中：这两年PaaS的活跃程度超过IaaS

在编排模块中：K8S目前是主流

在BI/Monitoring/Logging中：EFK之前用的比较多，普罗米修斯。
